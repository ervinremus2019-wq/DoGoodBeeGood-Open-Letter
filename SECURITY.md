# Security Policy

## Anti-Fraud & Security Measures

- **No SHA hashing on email addresses or identity fields** — all sender emails are stored and logged in full.
- **HMAC-SHA256** is used **only** for message signature verification, not for identity masking.
- **Full traceability** — every transaction, audit log, and receipt contains the full real sender email.
- **Strict allowlist** — only pre-approved sender emails are processed; all others are blocked and logged with full details.
- **Real-time fraud alerts** — any blocked or rejected attempt triggers an immediate alert to the system owner with the full sender email.
- **GDPR compliance** — data retention is strictly limited to 90 days unless required for fraud investigation.
